Class Y.Dispatcher

The Dispatcher class represents an object that can manage Node Elements to inject HTML content as the content of the Node..

Properties

Y.Dispatcher.ATTRS - static Object

Static property used to define the default attribute configuration of the component.

Methods

_dispatch

protected _dispatch ( content )
Dispatch a content into the code, parsing out the scripts, injecting the code into the DOM, then executing the scripts.
Parameters:
content <string> html content that should be injected in the page

_fetch

protected _fetch ( uri , cfg )
Fetching a remote file using Y.io. The response will be dispatched thru _dispatch method...
Parameters:
uri <string> uri that should be loaded using Y.io
cfg <object> configuration object that will be used as base configuration for Y.io (http://developer.yahoo.com/yui/3/io/#configuration)

stop

object stop ( )
Cancel the current loading and execution process immediately
Returns: object
reference for chaining

Configuration Attributes

autopurge - boolean

If dispatcher should purge the DOM elements before replacing the content
Default Value: true

content - string

default content for the dynamic area
Default Value: empty

loading - {boolean}

Boolean indicating that a process is undergoing.
Default Value: false

node - object

YUI Node Object that represent a dynamic area in the page.
Default Value: null

uri - string

URL that should be injected within the host
Default Value: null
