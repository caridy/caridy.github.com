Class Y.Dispatcher

The Dispatcher class represents an object that can manage Node Elements to inject HTML content as the content of the Node..

Properties

_io - protected Object

Connection Handler for AJAX requests.
Default Value: null

_queue - protected Object

Execution queue.
Default Value: null

Y.Dispatcher.ATTRS - static Object

Static property used to define the default attribute configuration of the component.

Methods

_dispatch

protected _dispatch ( content )
Dispatch a content into the code, parsing out the scripts, injecting the code into the DOM, then executing the scripts.
Parameters:
content <string> html content that should be injected in the page
Returns:
null

_executeScript

protected void _executeScript ( text , n )
Inject an inline script into the page as part of the dispatcher process.
Parameters:
text <string> Script code that should be executed
n <Node> A reference to the original SCRIPT tag Node, in case you want to get more specific attributes

_fetch

protected _fetch ( uri )
Fetching a remote file using Y.io. The response will be dispatched thru _dispatch method...
Parameters:
uri <string> uri that should be loaded using Y.io
Returns:
object Reference to the connection handler

_getScript

protected void _getScript ( src , n )
Inject an external script into the page as part of the dispatcher process. Due the async nature of this routine, we need to run the queue after the execution.
Parameters:
src <string> URI of the script that need to be injected
n <Node> A reference to the original SCRIPT tag Node, in case you want to get more specific attributes

_purgeContent

protected void _purgeContent ( )
Purge all the child node in preparation for a new content to be injected.

_setContent

protected void _setContent ( content )
Set a new content into the dispatcher host node.
Parameters:
content <string> HTML code that will replace the current content

stop

object stop ( )
Cancel the current loading and execution process immediately
Returns: object
reference for chaining

Configuration Attributes

autopurge - boolean

If dispatcher should purge the DOM elements before replacing the content
Default Value: true

content - string

default content for the dynamic area
Default Value: empty

ioConfig - Object

Default IO Config object that will be used as base configuration for Y.io calls. http://developer.yahoo.com/yui/3/io/#configuration
Default Value: null

loading - {boolean}

Boolean indicating that a process is undergoing.
Default Value: false

node - object

YUI Node Object that represent a dynamic area in the page.
Default Value: null

normalize - boolean

If dispatcher should analyze the content before injecting it. This will help to support full html document injection, to collect scripts and styles from head if exists, etc.
Default Value: false

uri - string

URL that should be injected within the host
Default Value: null
