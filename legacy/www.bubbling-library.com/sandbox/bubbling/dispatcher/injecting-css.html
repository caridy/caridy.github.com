CSS Injection Example - Dispatcher Plugin

Using the dispatcher plugin you will be able to inject CSS rules on demand in your application. The rules can be written in line or thru a remote file loaded thru AJAX using the YUI connection manager. See the source code to learn how to use the dispatcher plugin for the CSS injection.
