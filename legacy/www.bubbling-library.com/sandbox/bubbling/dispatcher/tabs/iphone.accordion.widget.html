Plugin Lighter - Example - Management your areas (YAHOO-CMS: YAHOO.plugin.Lighter)

Dynamic tab with an accordion control within...

This example is based on the lighter plugin ( YAHOO.plugin.Lighter)

Facts in this example:
- This content have autonomy.
- The dispatcher load the resources onDemand (YUI Animation, YUI Bubbling, YUI-CMS Lighter).
- The inline script block inside the content prepare the accordion control.
- The dispatcher inject the corresponding CSS (inline style tag).
- The dispatcher execute the script block to initialization the accordion.
